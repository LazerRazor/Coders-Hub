C++


C++ is a statically typed, compiled, general-purpose, case-sensitive, free-form programming language that supports procedural, object-oriented, and generic programming. C++ is regarded as a middle-level language, as it comprises a combination of both high-level and low-level language features.

Variables used in C++

A variable provides us with named storage that our programs can manipulate. Each variable in C++ has a specific type, which determines the size and layout of the variable's memory; the range of values that can be stored within that memory; and the set of operations that can be applied to the variable.

C++ If-Else Statements

An if-else statement controls conditional branching. Statements in the if-branch are executed only if the condition evaluates to a non-zero value (or true). If the value of condition is nonzero, the following statement gets executed, and the statement following the optional else gets skipped.
